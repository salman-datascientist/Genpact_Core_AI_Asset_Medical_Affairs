Who Is This For?

A Medical Affairs Director at GSK manages a cancer drug called Zejula (Niraparib). Their job:

  • Prove the drug works with real-world evidence (RWE)
  • Educate oncologists (HCPs — Healthcare Professionals) about the drug
  • Submit evidence packages to regulators (FDA, EMA) and payers (Aetna, NICE)

Every one of these tasks requires collecting data from multiple sources — manually, today. This BRD defines the AI system that will automate it.

The Manual "Stitching" Problem

📋 Request: "Show me outcomes for Zejula patients aged 55+ in the US, 2023"
Step 1 → Analyst logs into Optum portal Submits data extraction request Waits 2 weeks for Optum team to run the query Receives: claims_export_optum_2023.csv (300MB) Step 2 → Analyst logs into SEER Registry Downloads cancer registry export → seer_oc_patients_2023.sas7bdat Converts SAS → Excel manually Step 3 → Analyst goes to PubMed Searches: "Niraparib real world evidence ovarian cancer 2020-2024" Returns 847 papers → reads abstracts one by one Marks 40 as relevant → copy-pastes key data to Word Step 4 → Analyst opens all files side by side PROBLEM: Optum uses NDC drug codes, SEER uses ICD-O codes Analyst manually maps NDC → ICD codes using a reference table Step 5 → Creates summary PowerPoint → Sends to Medical Director ⏱ Total Time: 3–4 WEEKS for ONE request
🚨
CAUTION: The analyst is the "glue" between 5 incompatible systems. Remove the analyst and nothing works. This is not scalable for 450+ IEP requests per year.

Each Evidence Cycle Takes Weeks to Months

What is an Evidence Package (IEP)?

When Aetna (a US insurance company) considers whether to cover Zejula, they ask GSK:

💬 "Prove Zejula is more effective than Olaparib for platinum-resistant Ovarian Cancer in your US patient population"

GSK must submit an Integrated Evidence Plan (IEP) — a structured dossier containing:

  • Clinical trial data
  • Real-world effectiveness data
  • Cost-effectiveness analysis
  • Comparative effectiveness vs competitors
  • Geographic breakdowns

The Manual IEP Timeline (13 weeks)

Week 1–2
Analyst identifies data sources
Week 3–4
Data team extracts from IQVIA/Optum
Week 5–6
Statistician runs comparative analysis
Week 7–8
Medical writer drafts IEP document
Week 9–10
Medical Director review (round 1)
Week 11–12
Legal / MLR review — Veeva Vault
Week 13
Final submission to Aetna ✅
TOTAL: 13 WEEKS (3+ MONTHS) per evidence package
